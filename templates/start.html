Инструкция по использованию приложения

Добро пожаловать в наше приложение для расчетов на основе обученной нейронной сети LSTM! Здесь вы сможете загрузить данные для обучения, выполнить обучение нейросети и провести расчеты на основе введенных параметров. Следуйте данной инструкции для использования всех возможностей приложения.

Этапы работы с приложением

1. Загрузка данных для обучения

Чтобы обучить нейросеть на ваших данных, сначала необходимо загрузить CSV файл с данными. Для этого:

  • Перейдите на вкладку "NET".
  • Нажмите кнопку "Выбрать файл" и выберите CSV файл с данными.
  • Файл должен содержать следующие столбцы:
    • total_kdsi: Количество строк исходного кода.
    • aaf: Adjusted Application Factor.
    • rely: Надежность программного обеспечения.
    • data: Размер базы данных.
    • cplx: Сложность программного обеспечения.
    • time: Временные ограничения.
    • stor: Требования к памяти.
    • virt: Виртуальные машины.
    • turn: Время отклика системы.
    • acap: Способности аналитиков.
    • aexp: Опыт аналитиков.
    • pcap: Способности программистов.
    • vexp: Опыт использования виртуальных машин.
    • lexp: Опыт в разработке на языках программирования.
    • modp: Использование современных методов разработки.
    • tool: Использование инструментов разработки.
    • sced: Соблюдение сроков.
    • effort: Трудозатраты в человеко-месяцах (целевое значение).
  • После выбора файла нажмите "Загрузить".
  • Если данные были успешно загружены, вы увидите уведомление об успешной загрузке.

2. Обучение нейросети

После загрузки данных вы можете начать обучение нейросети:

  • Перейдите на вкладку "Обучение".
  • На открывшейся странице нажмите кнопку "Запустить обучение".
  • Во время обучения вы будете видеть прогресс в виде обновляющегося графика с потерями на обучении и валидации.
  • Подождите завершения обучения — по его окончании модель будет готова для расчетов.

3. Проведение расчетов

Когда нейросеть обучена, вы можете вводить данные для расчета:

  • Перейдите на вкладку "Расчет".
  • Заполните поля в форме, указав данные, по которым вы хотите провести расчет.
  • После заполнения всех полей нажмите кнопку "Рассчитать".
  • На экране отобразится результат расчета — предсказанное значение трудозатрат (effort).

4. Просмотр загруженных данных

Если вы хотите просмотреть загруженные данные, перейдите на вкладку "Загруженные данные". На этой странице вы увидите таблицу с загруженными ранее данными.

5. Просмотр данных компании

Для просмотра основных данных о компании перейдите на вкладку "Данные компании".

6. Вход и регистрация

Для работы с приложением вам может понадобиться зарегистрироваться или войти в свой аккаунт. Кнопки для входа и регистрации расположены в правом верхнем углу экрана.

Требования к файлам и данным

Ваш CSV файл должен быть в формате с разделителем точка с запятой (;), а все числовые значения должны быть в формате десятичных чисел.

Дополнительная информация

Если у вас возникли вопросы или проблемы при работе с приложением, пожалуйста, свяжитесь с нашей поддержкой.
